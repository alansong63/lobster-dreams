# 树莓派 Raspberry Pi 产品线演进

## 系列总览

| 系列 | 定位 | 代表型号 |
|------|------|----------|
| **Pi 1** | 入门学习 | Model B, Model B+, Model A, Model A+ |
| **Pi 2** | 性能升级 | Model B |
| **Pi 3** | 无线化 | Model B, Model B+, Model A+ |
| **Pi 4** | 桌面级 | Model B |
| **Pi 5** | 旗舰性能 | Model B |
| **Zero** | 超小巧 | Zero, Zero W, Zero 2 W |
| **Compute Module** | 工业嵌入式 | CM1, CM3, CM3+, CM4, CM5 |

---

## Pi 1 系列 (2012-2014)

### Model B (2012)
- CPU: 700MHz 单核 ARM11
- RAM: 256MB / 512MB
- 接口: USB 2.0×2, HDMI, 100Mbps 以太网, 26针 GPIO
- 价格: $25

### Model B+ (2014)
- CPU: 700MHz 单核 ARM11
- RAM: 512MB
- 接口: USB 2.0×4, HDMI, 100Mbps 以太网, 40针 GPIO
- 特点: 最终版，增加USB口和GPIO

### Model A+ (2014)
- CPU: 700MHz 单核 ARM11
- RAM: 256MB / 512MB
- 接口: USB 2.0×1, HDMI, 无以太网, 40针 GPIO
- 价格: $20

---

## Pi 2 系列 (2015)

### Model B (2015)
- CPU: 900MHz 四核 ARM Cortex-A7
- RAM: 1GB
- 接口: USB 2.0×4, HDMI, 100Mbps 以太网, 40针 GPIO
- 特点: 首次四核处理器

---

## Pi 3 系列 (2016-2018)

### Model B (2016)
- CPU: 1.2GHz 四核 ARM Cortex-A53 (64位)
- RAM: 1GB
- 无线: Wi-Fi 802.11n, 蓝牙 4.1
- 接口: USB 2.0×4, HDMI, 100Mbps 以太网
- 特点: 首次内置无线

### Model B+ (2018)
- CPU: 1.4GHz 四核 ARM Cortex-A53 (64位)
- RAM: 1GB
- 无线: Wi-Fi 802.11ac, 蓝牙 4.2
- 接口: USB 2.0×4, HDMI, 千兆以太网
- 特点: 升级无线和网口

### Model A+ (2018)
- CPU: 1.4GHz 四核 ARM Cortex-A53
- RAM: 512MB
- 无线: Wi-Fi 802.11ac, 蓝牙 4.2
- 接口: USB 2.0×1, HDMI, 无以太网
- 价格: $25

---

## Pi 4 系列 (2019)

### Model B
- CPU: 1.5GHz 四核 ARM Cortex-A72 (64位)
- RAM: 2GB / 4GB / 8GB
- 接口: USB 3.0×2, USB 2.0×2, 千兆以太网, 双 micro-HDMI
- 无线: Wi-Fi 802.11ac, 蓝牙 5.0
- 特点: 性能飞跃，可运行完整桌面系统

---

## Pi 5 系列 (2023)

### Model B
- CPU: 2.4GHz 四核 ARM Cortex-A76
- RAM: 4GB / 8GB
- 接口: USB 3.0×2, USB 2.0×2, 千兆以太网, 双 micro-HDMI, PCIe 2.0
- 无线: Wi-Fi 802.11ac, 蓝牙 5.0
- 特点: 性能比 Pi 4 提升 2-3 倍

## Zero 系列 (2015-2021)

### Zero (2015)
- CPU: 1GHz 单核 ARM11
- RAM: 512MB
- 接口: micro-USB×1, mini-HDMI, 40针 GPIO (排针需另购)
- 特点: 最小巧，$5

### Zero W (2017)
- CPU: 1GHz 单核 ARM11
- RAM: 512MB
- 无线: Wi-Fi 802.11n, 蓝牙 4.1
- 价格: $10

### Zero 2 W (2021)
- CPU: 1GHz 四核 ARM Cortex-A53
- RAM: 512MB
- 无线: Wi-Fi 802.11n, 蓝牙 4.2
- 价格: $15

## Compute Module 系列

### CM1 (2014)
- SoC: BCM2835
- RAM: 512MB
- 存储: 无 (需外接)

### CM3 / CM3+ (2017/2019)
- SoC: BCM2837
- RAM: 1GB / 4GB (CM3+)
- 存储: 8GB eMMC (可选)

### CM4 (2020)
- SoC: BCM2711
- RAM: 1GB / 2GB / 4GB / 8GB
- 存储: 可选 eMMC, Wi-Fi, Bluetooth
- 接口: 双 HDMI, MIPI, PCIe

### CM5 (2024)
- SoC: BCM2712
- RAM: 2GB / 4GB / 8GB / 16GB
- 存储: 可选 eMMC
- 特点: 最新旗舰计算模块
